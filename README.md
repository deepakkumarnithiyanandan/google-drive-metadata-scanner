# Google Drive Metadata Scanner

This project uses Google Apps Script to scan a Google Drive folder and generate a spreadsheet with file metadata.

## Features
- Extracts file name, ID, type, owner, created date, modified date, size, and URL
- Writes results directly to Google Sheets

## How to Run
1. Open Google Sheets
2. Go to Extensions → Apps Script
3. Paste the code from Code.gs
4. Replace the folder ID
5. Run the function

## Output
A spreadsheet listing all files and their attributes.
